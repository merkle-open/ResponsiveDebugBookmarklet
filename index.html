Browser size, Breakpoint, Image size

The Namics Responsive Debug Bookmarklet is a bookmarklet displaying an information bar with browser size, and (in the future) highlighting of different breakpoints.
Additionally you can activate overlays with the image dimensions of each inline image.


Functionality

Breakpoint

The breakpoint highlighting functionality is planned at the moment. You will be able to choose different breakpoints or even define own ones (saved in local storage). The breakpoints will be displayed in the top bar.

Image size

The image sizes overlays are color coded to make it easy to identify a potential problem.

A yellow background indicates a picture which is scaled down by the browser, if it is scaled down to much you should check if the file size gets to big for lower bandwidth.
You should consider to load different images for specific browser sizes.
A red background indicates a picture which is scaled up by the browser. This should not happen at all because the resolution of the picture is to low and it gets blurry.

Installation

Just drag the following link to your favorites bar to "install" the bookmarklet:
Namics Responsive Debug Bookmarklet
To activate the bookmarklet click onto the new favorites bar entry when visiting the site you want to inspect.

Note

Please be aware that the bookmarklet adds an script hosted on github, additionally jQuery is loaded from the Google CDN so you have to be online in order to use this script.
If you like to use the bookmarklet offline feel free to download the script, add it to your local webserver and change the urls.
